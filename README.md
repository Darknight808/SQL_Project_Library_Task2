# 📚 Library Management System – SQL Project

## 📌 Project Overview
This project is a SQL-based analysis of a Library Management System database.  
The goal is to explore and analyze data related to books, members, and borrowing activities using SQL queries.

This project helped me strengthen my skills in querying, data analysis, and working with relational databases.

---

## 🛠️ Tools & Technologies
- PostgreSQL
- pgAdmin 4
- SQL

---

## 📂 Dataset Description
The dataset simulates a library system and includes tables such as:

- Books
- Members
- Borrowings
- Authors

---

## 🔍 Key Tasks Performed
- Data exploration using SELECT queries
- Filtering data using WHERE conditions
- Aggregations using COUNT, SUM, AVG
- Grouping data using GROUP BY
- Joining multiple tables using JOINs
- Sorting and organizing results

---

## 📊 Example Analyses
Some of the questions explored in this project:

- Which books are borrowed the most?
- How many books are borrowed per member?
- What are the most active members?
- Trends in borrowing activity
